SETUP GUIDE

RUNNING IN
20 MINUTES.

Web app on Vercel, mobile app via Expo QR on every scout's phone. Follow these steps exactly.

1
Create Supabase Project

Go to supabase.com → New Project. Pick a region close to your competition. Copy your Project URL and anon key from Settings → API.

Open Supabase →
2
Run the Schema + Seed

In Supabase → SQL Editor → New Query. Paste supabase/migrations/001_initial_schema.sql, click Run. Then run supabase/seed.sql for test data.

Order matters: Schema first (creates tables), seed second (inserts test data). The seed loads 20 real FTC teams and a test event so everything works immediately.
3
Get Your API Keys
ANTHROPIC_API_KEY
console.anthropic.com → API Keys
Server-side only — never in Expo bundle
FTC_API_KEY + FTC_API_SECRET
ftc-events.firstinspires.org → Account → API
Sync rankings and match results
NEXT_PUBLIC_SUPABASE_URL + ANON_KEY
Supabase → Settings → API → Project URL + anon key
Safe to expose to client
4
Deploy to Vercel

Push the repo to GitHub, import in Vercel. It auto-detects Next.js in apps/web. Add all env vars in Vercel → Settings → Environment Variables.

npm install -g turbo npx create-turbo@latest coolfTC # copy your files into the scaffold cd apps/web && vercel deploy
5
Start the Expo Dev Server

From apps/mobile, create a .env.local with your Supabase keys and the Vercel URL, then start Expo. A QR code appears in your terminal.

cd apps/mobile # Create .env.local with: # EXPO_PUBLIC_SUPABASE_URL=https://xxx.supabase.co # EXPO_PUBLIC_SUPABASE_ANON_KEY=eyJ... # EXPO_PUBLIC_API_BASE_URL=https://your-app.vercel.app npx expo start
Competition day tip: If scouts are on cellular (not same WiFi), use npx expo start --tunnel — the QR works from anywhere.
6
Create Team + Invite Scouts

Go to your web app → Create Team. Set org name and FTC team number. You're auto-admin. Then Settings → Members to invite scouts — each gets an account and joins your org.

7
Set Your Event Key + Sync Data

Every page has an event key field. Format: YYYY-EVENTCODE. Go to Settings → Sync FTC Data to pull rankings for your event. The seed uses 2025-DECODE-TEST for instant testing.

# Example event keys: 2025-TXHOU # Texas Houston qualifier 2025-TXDAL # Texas Dallas qualifier 2025-DECODE-TEST # Seed data, works immediately
